### Install from source (recommended for Rust users)

```bash
# Clone and build
git clone https://github.com/BitPusher16/qwench.git
cd qwench
cargo install --path .

# Or one-liner (no clone needed)
cargo install --git https://github.com/BitPusher16/qwench.git
